# raspberrypi-sppech-recognition
<b>概要:</b>

英語で時刻を尋ねると答えるシステムです。
”What time is it?”と言ってください。
終了するには”Good bye.”と言ってください。


<b>ハードウェア：</b>

Raspberry PI3 model B

USB マイク

3.5mm audio jack スピーカー


<b>Python：</b>

ver 3.9.2
